# MC Nylon Experiment Skill

当用户要求设计浇铸尼龙实验、优化工艺路线、计算投料、安排变量、整理实验记录表、
规划下一轮试验时，使用这个 skill。

## 解决什么问题

- 把实验想法整理成可执行 SOP
- 把配方思路转成变量矩阵和批次计划
- 把摩尔比、质量比、每 100 mol caprolactam 的用量表达统一
- 明确记录哪些工艺窗口必须被追踪
- 为后续缺陷分析保留足够实验记录

## 什么时候触发

- “帮我设计一轮 MC 尼龙实验”
- “算一下 catalyst / activator 的投料”
- “给我一个记录表”
- “我想比较不同脱水条件/真空条件”
- “填料应该什么时候加”
- “下一轮实验怎么缩小变量范围”

## 建议输入

1. 实验目标
   - 增强、增韧、耐磨、低缺陷、低残余单体等
2. 原料与体系
   - caprolactam、catalyst、activator、TDI、NaOH、glass fiber、silica 等
3. 已知工艺范围
   - 熔融、一次脱水、二次脱水、真空、加入温度、浇铸、模具、聚合、保温
4. 约束条件
   - 设备能力、真空上限、模具类型、批量、是否能做水浴/油浴
5. 输出用途
   - 直接做实验、整理记录模板、准备小试或中试

## 默认工作方式

1. 先明确实验目标和主变量。
2. 列出必须控制的关键节点：
   - caprolactam 熔融
   - 一次脱水温度/时间
   - 真空度
   - catalyst 加入温度
   - 二次脱水温度
   - activator 加入温度
   - 浇铸温度
   - 模具温度
   - 聚合温度/时间
   - 保温温度/时间
   - 水浴/油浴条件
   - 填料加入时机与分散方式
3. 如涉及配比，优先用用户提供或文献明确给出的数值。
4. 若用户要求换算，只在已知分子量、目标比例和原料定义清晰时计算。
5. 输出 SOP、记录表、变量矩阵和风险点。

## 默认输出格式

```text
实验目标
- ...

实验变量与固定条件
- ...

建议 SOP
- ...

投料与换算说明
- ...

批次记录模板
- ...

风险点检查表
- ...

下一轮实验建议
- ...

信息缺口与需确认项
- ...
```

## 不能做什么

- 不编造温度、时间、真空度、摩尔比、投料量
- 不在原料定义不清时强算质量比
- 不把文献中的局部条件当成通用最佳值
- 不跳过安全、设备限制和记录要求
- 不把猜测性的机理解释替代为工艺结论

## 不确定时如何处理

- 若输入不足，先给“需要确认的字段清单”，再给结构化模板
- 若用户要计算投料，但缺少分子量、纯度、目标批量或比例定义，先停止计算并说明缺口
- 若工艺窗口来源不明确，标注“待实验确认”
- 若多变量过多，优先建议一轮只改 1 到 2 个主变量

## 输出风格要求

- 中文为主，保留必要英文术语和化学缩写
- 优先输出可执行表格或清单
- 先分“已知条件”和“待确认条件”
- 任何定量建议都要标注其依据；无依据时只给框架不给数值
