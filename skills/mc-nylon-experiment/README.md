# mc-nylon-experiment

## 用途

用于围绕 MC 尼龙制备与改性设计实验方案、变量矩阵、投料换算和实验记录模板。

## 适用场景

- 设计一轮新的浇铸尼龙实验
- 比较脱水、真空、加入顺序、模具温度等变量
- 计算催化剂/活化剂或填料投料
- 建立批次记录表和风险检查表

## 输出

- 实验 SOP
- 批次记录模板
- 变量矩阵
- 投料计算表
- 风险点检查表
- 下一轮实验修改建议

## 文件结构

- `SKILL.md`：主流程和判断边界
- `agents/openai.yaml`：Codex 展示信息
- `references/terminology.md`：实验术语与换算口径
- `references/checklist.md`：SOP 与记录检查表
- `references/templates.md`：SOP、批记录、变量矩阵模板

## 注意事项

- 这个 skill 不应在缺少必要输入时编造数值
- 所有计算都应回到“输入是否充分、定义是否明确”
- 变量矩阵应尽量控制复杂度，避免一轮实验改太多因素
