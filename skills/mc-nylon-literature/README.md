# mc-nylon-literature

## 用途

用于 MC 尼龙、PA6 铸型尼龙、己内酰胺阴离子聚合、增强增韧、耐磨和工艺变量相关的文献检索与研究方向梳理。

## 适用场景

- 查找某个变量的代表性论文
- 比较 glass fiber、silica、nano-silica 等增强方案
- 梳理 catalyst / activator / TDI / NaOH 相关文献
- 建立 Zotero 分类结构
- 找研究空白、争议点和实验启发

## 输出

- 文献清单
- 主题分组
- 检索关键词
- 数据库检索式
- Zotero 分类建议
- 争议点与研究空白
- 对实验设计的启发

## 文件结构

- `SKILL.md`：主工作流和输出规范
- `agents/openai.yaml`：Codex 展示信息
- `references/terminology.md`：中英术语和缩写
- `references/keywords.md`：关键词与检索式素材
- `references/checklist.md`：检索与筛选检查表
- `references/templates.md`：输出模板

## 注意事项

- 这个 skill 不应编造文献和性能结果
- 如果没有实际联网检索，要明确说明当前输出是检索方案或整理框架
- 对实验值只提供“建议核查的变量”，不直接伪造定量结论
