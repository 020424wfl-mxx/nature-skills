# 缺陷与工艺术语

- bubble：气泡
- pore / void：孔洞
- whitening：发白
- yellowing：黄变
- brittle fracture：脆裂
- flow mark：流痕
- delamination：分层
- short shot / insufficient fill：缺料/充模不足
- shrinkage：收缩
- agglomeration：团聚
- sedimentation：沉降
- poor dispersion：分散不良
- incomplete polymerization：聚合不完全
- residual monomer：残余单体
- moisture issue：含水问题
- insufficient vacuum：真空不足
- ratio deviation：配比偏差
- temperature control deviation：温控异常
