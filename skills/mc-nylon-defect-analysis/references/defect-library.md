# 常见缺陷库

## 气泡 / 孔洞

常见方向：

- 含水控制不足
- 真空不足或真空阶段不稳定
- 浇铸过程卷入空气
- 反应放热与流动窗口不匹配

验证重点：

- 回看脱水、真空、浇铸连续性
- 比较不同批次的外观分布
- 检查缺陷是否集中在厚壁或末端区域

## 发白 / 脆裂

常见方向：

- 聚合不完全
- 内部缺陷导致应力集中
- 填料分散差或界面结合差
- 后处理或测试条件不一致

验证重点：

- 看断口特征
- 核查残余单体和成型均匀性
- 对照成功批次的配方与工艺

## 黄变

常见方向：

- 温度过高或保温过久
- 原料状态不稳定
- 某些添加剂或体系副反应

验证重点：

- 核查热史
- 比较不同批次颜色变化时间点
- 区分表面黄变和整体黄变

## 团聚 / 沉降 / 分散不良

常见方向：

- 填料加入时机不合适
- 混合能量不足
- 体系黏度窗口不适合当前填料
- 填料表面状态不匹配

验证重点：

- 比较加入顺序和混合时间
- 检查样品不同位置填料分布是否一致
- 记录流动性变化和沉降趋势

## 聚合不完全 / 无法成型

常见方向：

- catalyst / activator 配比异常
- 含水问题
- 温控异常
- 原料处理或加入顺序问题

验证重点：

- 核查配比定义和称量记录
- 对照温度时间记录
- 检查原料预处理是否一致
