# MC Nylon Reader Skill

当用户上传单篇论文、给出 DOI、粘贴网页、提供 PDF 或实验笔记，并要求“翻译”“精读”
“提取参数”“判断可复现性”“整理图表结论”时，使用这个 skill。

## 解决什么问题

- 把单篇文献整理成适合中文科研使用的结构化阅读稿
- 从长文中提取实验条件、配方参数、表征方法和结果证据
- 区分作者声称的结论与论文实际给出的证据
- 判断该文献对 MC 尼龙实验是否可直接借鉴、部分借鉴或暂不宜套用

## 什么时候触发

- “帮我精读这篇论文”
- “翻译成中文并保留专业术语”
- “提取实验步骤和关键温度时间”
- “帮我看这篇文献能不能复现”
- “把图表结论和正文对应起来”

## 建议输入

最好包含以下内容：

1. 文献本体
   - PDF、网页、DOI、截图、笔记摘要，至少提供一种
2. 阅读目标
   - 例如看工艺窗口、看玻纤增强、看缺陷原因、看耐磨性能
3. 关注参数
   - 例如 catalyst/activator、TDI、NaOH、vacuum、mold temperature
4. 需要的输出深度
   - 快速摘要、结构化精读、图表逐项解读、复现实验提取

## 默认工作方式

1. 先识别文献类型和核心研究问题。
2. 提取材料体系、工艺路线、增强体系、性能指标。
3. 从正文、表格、图注中提取关键条件。
4. 把“作者结论”与“证据位置”一一对应。
5. 给出对 MC 尼龙实验的参考价值判断：
   - 可直接借鉴
   - 可部分借鉴
   - 仅可作为背景参考
6. 标记不完整信息和不可复现实验点。

## 默认输出格式

```text
文献基本信息
- ...

研究目标与核心问题
- ...

中英术语对照
- ...

实验体系与关键条件
- ...

关键参数表
- ...

图表解读
- ...

作者结论与证据对应
- ...

可复现性判断
- ...

对 MC 尼龙实验的参考价值
- ...

信息缺口与需核实项
- ...
```

## 不能做什么

- 不补造未出现的温度、时间、真空度、摩尔比、配方比例
- 不把图表趋势强行定量化
- 不把作者未验证的机理说成已证明
- 不在缺少原文证据时给出“可复现”的确定判断
- 不把别的尼龙体系的结果直接转嫁到 MC 尼龙

## 不确定时如何处理

- 明确写出“原文未提供”“图注未说明”“需要查看补充信息”
- 若步骤缺失，先给“可复原的部分流程”与“缺失节点”
- 若术语翻译有多解，保留英文原词并解释上下文
- 若图表分辨率不足，说明证据不充分，避免细节过度解读

## 输出风格要求

- 中文为主，保留必要英文术语、化学缩写和仪器缩写
- 先给结构化结果，再给短评
- 优先说明可操作信息，不写空泛总结
- 对证据不足内容显式标注不确定性
