# mc-nylon-reader

## 用途

用于把单篇 MC 尼龙相关文献、PDF、网页或笔记整理成中文科研阅读稿，并提取实验和图表信息。

## 适用场景

- 上传一篇论文后要求精读或翻译
- 提取实验步骤、配方参数和工艺条件
- 判断文献是否具备可复现性
- 把图表结论和正文证据对应起来

## 输出

- Markdown 阅读稿
- 关键参数表
- 图表解读
- 实验流程复原
- 结论与证据对应表
- 可借鉴点与不可直接套用点

## 文件结构

- `SKILL.md`：主流程和输出规范
- `agents/openai.yaml`：Codex 展示信息
- `references/terminology.md`：术语与翻译口径
- `references/checklist.md`：阅读与复现检查表
- `references/templates.md`：阅读稿和参数表模板

## 注意事项

- 这个 skill 不应填补原文没有给出的参数
- 对图表证据必须注明来源位置或说明“原文未明确”
- 判断可复现性时要区分“步骤完整度”和“机理可信度”
